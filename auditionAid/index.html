Audition Aid

A tool for discovering better songs for vocal auditions.

Description

Audition Aid was created by Maggie Hewitt, Melissa Young, Paige Satterly, and Leslie Bowen for RIT's WiC Hack's 2015. It was designed to be a tool for finding the perfect audition piece for musicals. Users would select a musical and a role similiar to their vocal range. Audition Aid would then recommend pieces in a similair vocal range. Recommendations could be played via an integrated Spotify player and included a link to the song on the Google Play Store. Audition Aid was written in Python using the microframework Flask.

Awards: Recipient of the Best Microsoft Hack Award - Wic Hacks 2015
